# HRIS Frontend Application

A production-ready Human Resource Information System frontend built with React, Vite, and Tailwind CSS.

## Features

- Role-based access control (Applicant, Employee, Manager, HR, Executive)
- Recruitment & Onboarding workflow
- Talent Application Portal
- Employee Dashboard
- Offboarding Management
- Manager/Executive Dashboards with analytics

## Tech Stack

- React 18
- Vite
- React Router
- Tailwind CSS
- Recharts

## Installation

```bash
npm install
npm run dev
```

## Design System

Uses a warm, premium design system with:
- Warm ivory/cream base backgrounds
- Ambient mesh gradients
- Soft beige, peach, sand, and amber tones
- Modern, enterprise-grade UI
